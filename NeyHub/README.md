# Ney Hub UI Library

Fork of the Obsidian UI Library, rebranded and organized as the base for all of Ney's scripts.

End users load scripts through the single loader at the repo root ([`Loader.lua`](../Loader.lua)), which checks the game's PlaceId and only runs a script for games Ney Hub actually supports.

**Upstream documentation (still applies, just mentally swap "Obsidian" for "Ney Hub"):** https://docs.mspaint.cc/obsidian

**Upstream source code to documentation site:** https://github.com/mspaint-cc/docs.mspaint.cc/tree/main/content/obsidian
